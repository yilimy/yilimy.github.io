20180202添加评论gitment

发表于 2018-02-02 | 阅读次数

简介

国内的评论需要实名，需要备案，emmm…好在imsun实现的一款基于 GitHub Issues 的评论系统Gitment，很是厉害的。之后，yanzi1225627又对其做了改进，使其不用在针对性添加评论标签(作者文档上说在需要的页面上添加)，而是统一配置于每一页面。评论支持md，比较适合程序员。
Gitment 是作者实现的一款基于 GitHub Issues 的评论系统。支持在前端直接引入，不需要任何后端代码。可以在页面进行登录、查看、评论、点赞等操作，同时有完整的 Markdown / GFM 和代码高亮支持。尤为适合各种基于 GitHub Pages 的静态博客或项目页面。
Gitment 只能使用 GitHub 账号进行评论。
