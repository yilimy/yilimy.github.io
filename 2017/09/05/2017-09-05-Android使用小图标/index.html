Android使用小图标

发表于 2017-09-05 | 阅读次数
Font Awesome代替图标

传送门：Font Awesome官网 传送门：with Unicode (Cheatsheet)

传统显示图标的方式

在平常的开发中，如果我们需要在界面上显示某个小图标，比如搜索按钮，返回按钮，这时我们需要美工给我们切对应的png图片，并放进对应的drawable文件夹中，这样随着图标的越来越多，APK体积也会越来越大。
或者，我们可以使用android Draw来绘制自己的图片，但是，这种方法的局限性很大，android提供的绘图API支持简单的图像绘制， 比如矩形、圆形、钢笔描线，太精细的图标可能不尽如人意。

什么是Font Awesome

Font Awesome是一个专为Bootstrap设计的字体文件，我们可以通过向显示字体一样方便的显示我们想要显示的图标。对于Android，Font Awesome使用TextView显示小图标，而且无需使用png图片，只需使用一个ttf文件，并且在这些字体中的图片都是矢量图，是可以放大和缩小的，这就意味着每个图标都能在所有大小的屏幕上完美呈现。
Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS.

如何使用

  1. 下载Font Awesome，格式ttf
  2. 把下载后的ttf放到assets文件夹中
  3. 编写string.xml,可以在http://fontawesome.io/cheatsheet/# 中查看图标对应的字符串(Unicode码)
  4. android:text引用你要显示的图标对应的string资源，android:textColor设置图标颜色，android:textSize设置图标大小（跟设置字体大小颜色一样样的）
