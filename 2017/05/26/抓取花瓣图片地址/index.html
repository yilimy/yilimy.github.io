抓取花瓣图片地址

发表于 2017-05-26 | 阅读次数
使用safari抓取花瓣画板中图片的真实地址

使用Safari打开花瓣中某一个图片，eg:http://huaban.com/pins/668155974/
猜想花瓣网使用的策略类似于pixiv(P站)，可以通过图片的ID找到对应的图片资源。
故，“668155974”应该是与图片的ID相关。目前是要找到图片的真实路径，因此，要对这个ID中的信息进行分析。
开启Safari网页检查器，重点关注javascript部分，如下图：

在app.page对象中查找pins属性，但是这部分javascript又多又杂。
不妨尝试切换窗口“元素”->“资源”，在“资源”窗口尝试查找：

pins里面存放的是一组图片信息，可能与分类有关，这里暂不做分析。
搜索图片ID：668155974就可以找到pins中的那一项对应我们需要的图片信息了。

然后通过分析属性 -> file -> key来获取真实的图片路径，上面信息中的key是f20bd3c758ab3b86698951560d9714b6fa88dffd64c01-LAj5IT
图片地址：
http://img.hb.aicdn.com/f20bd3c758ab3b86698951560d9714b6fa88dffd64c01-LAj5IT

为检测key是否与用户或者session相关：

  • 注销登录后重新访问当前网址，能获取图片，成功
  • 在其他设备上访问当前网址，能获取图片，成功

尚且，可将链接作为图片静态地址使用
