4-1 创建list

1.假设班级里有3名同学：Adam,Lisa和Bart,他们的成绩分别是95.5,85,和59，请按照名字，分数，名字，分数…的顺序按照分数从高到低用一个list表示，然后打印出来

4-2 按照索引访问list

三名同学的成绩可以用一个list表示：
L = [95.5,85,59]
请按照索引分别打印出第一名、第二名、第三名、第四名的分数。

4-3 倒序访问list
