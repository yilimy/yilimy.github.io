3-9布尔类型

请运行如下代码，并解释打印的结果：
a = 'Python’
print ‘Hello,’, a or 'world’
b = ''
print ‘Hello,’, b or 'world’

Python把0、空字符串和None看成是False，其他数值和非空字符串都看成是True
