ERROR:gen already exists but is not a source folder

发表于 2017-09-01 | 阅读次数

更新SVN后，project编译不通过，出现错误：
gen already exists but is not a source folder. Convert to a source folder or rename it. 解决方案： 按照提示， 1.右键点击Package Exploreer下的myProject，选择Properties.

2.选择Java Build Path,再选择Source.

3.点击右边的Add Folder, 选中gen文件夹，点击OK，之后Yes.

4.在Properties的主面板中点击OK确定。

5.右键点击Package Exploreer下的myProject，选择Android Tools, 点击Fix Project Properties .
